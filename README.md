# ComicArc

A local comic reader for files you own. No accounts, no cloud, no subscriptions.

> Only import files you legally own or have the right to access. See [LEGAL.md](LEGAL.md).

---

## Quick Start

**macOS (recommended):**
1. Download **ComicArc.app** from [Releases](https://github.com/Vivekmurugulla2004/ComicArc/releases)
2. Move it to your Applications folder
3. Double-click to open — a setup wizard walks you through everything

That's it. No Python, no terminal, no configuration required.

---

## First Launch

The first time you open ComicArc, a short wizard guides you through:

1. **Choose your library folder** — pick the folder where your comic files live
2. **Scan** — ComicArc imports everything automatically
3. **Pick a reading mode** — page-by-page or scroll view

After setup, ComicArc opens straight to your library on every launch and automatically rescans for new files.

## What This Is

ComicArc is a **single-user, local-first** desktop app. It organizes and reads comic files that live on your hard drive — it does not host, stream, or distribute content.

Think of it the same way you think about:
- **Music** — organizes music files you own
- **Infuse** — organizes video files you own
- **Calibre** — organizes ebook files you own

ComicArc does the same thing for comic files.

## Features

**Library**
- Drag-and-drop or folder import for CBZ, CBR, PDF, and image files
- Grid view with cover thumbnails, reading progress, and star ratings
- Filter by publisher, tags, or search by title/series
- Favorites and Continue Reading section
- Reading list ("Want to Read") to queue up issues
- Bulk actions — mark read/unread, add to reading list, delete multiple at once
- Metadata editor (title, series, publisher, issue number)
- Manual drag-and-drop reordering

**Reader**
- Page-by-page and vertical scroll (manga) reading modes
- Double-page spread mode
- Zoom and pan
- Autoplay mode — automatically advances pages on a timer
- Keyboard shortcuts (see table below)
- Touch swipe support
- Progress saves automatically

**Narrative Runs**
- Build ordered reading lists spanning multiple series and publishers
- Drag-and-drop reordering
- Per-issue notes, ratings, and favorites
- Auto-advance between comics in a run

**Stats**
- Total comics, pages read, favorites, completion tracking
- Breakdown by publisher and top series
- Recently read history

**Settings**
- Change library folder and rescan anytime
- Switch default reader mode (page / scroll)
- Install CBR support (unar) via one click on macOS
- Export full library data as JSON backup
- Reset onboarding wizard without losing library data
- Clear library (files stay on disk)

## Keyboard Shortcuts (Reader)

| Key | Action |
|-----|--------|
| `←` / `→` | Previous / next page |
| `Space` | Next page |
| `A` | Toggle autoplay |
| `V` | Vertical scroll mode |
| `D` | Double-page spread |
| `Z` | Zoom |
| `F` | Fullscreen |
| `Escape` | Exit zoom / close modal / stop autoplay |
