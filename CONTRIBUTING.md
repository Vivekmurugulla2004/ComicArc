# Contributing

ComicArc is a personal project. Pull requests are not accepted.

If you find a bug, you're welcome to [open an issue](https://github.com/Vivekmurugulla2004/ComicArc/issues/new/choose). Please include:

- Your macOS version (e.g. macOS 14.5 Sonoma)
- What you did, what you expected to happen, and what actually happened
- Any error messages or unusual behavior you noticed
- The comic format involved, if relevant (CBZ, CBR, PDF, etc.)

Feature requests are tracked but not guaranteed. ComicArc evolves at its own pace.
